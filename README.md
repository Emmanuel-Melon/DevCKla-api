# DevCKla-api

devckla.github.io api.

## Getting Started

- Clone the repository

`git clone https://github.com/devckla/DevCKla-api.git`

- Then

`cd DevCKla-api`

- Run `mv .env.template .env`

- Edit the `.env` file to replace the values with your own configs

- Run `npm install`

- Finally `npm start`
